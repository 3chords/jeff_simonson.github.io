welcome to the.data.dive!

My name is Jeff Simonson. I am a data scientist with anemic HTML skills.

In this episode of the.data.dive I reveal some findings from a recent study of randomly selected cities using the OpenWeatherMap API.

For this analysis I used libraries like Python Pandas and Matplotlib to investigate whether a city's degrees of north/south latitude has any interesting relationships with temperature, humidity, cloudiness and wind speed. To accomplish this we access the OpenWeatherMap API to randomly select 500 global cities based on latitude and then plot our results against these key data points. All data sampled for this analysis is from May 26, 2019.
